# SCT_DS_3
# Decision Tree Classifier - Bank Marketing Dataset

## Overview

This project builds a Decision Tree Classification model to predict whether a customer will subscribe to a term deposit based on the Bank Marketing dataset.

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Tasks Performed

* Loaded and explored the dataset.
* Encoded categorical variables using LabelEncoder.
* Split the dataset into training and testing sets.
* Built a Decision Tree Classifier.
* Evaluated the model using accuracy and classification report.
* Visualized the Decision Tree.

## Output

* Decision Tree Visualization
* Model Accuracy
* Classification Report

## Dataset

Bank Marketing Dataset from the UCI Machine Learning Repository.
